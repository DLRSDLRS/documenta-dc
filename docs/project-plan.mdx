## General View

- Ein Monorepo (Turborepo) für ein Design System erstellen
    - Unterschiedliche Typescript/Javascript NPM Packages erstellen
    - Linting und Testing
    - Versionierung mit Changesets
    - UI Komponenten/Packages erstellen
        - Buttons, Containers, Inputs/Labels
        - Die Library vanilla-extract wird für die Erstellung von Styles
                benutzt (CSS)
        - CSS Classes müssen mit Vanilla JS funktionieren
        - Extra Komponenten für Vue und React
    - Storybook Library für Visualisierung und Testen von UI Komponenten
    - Dokumentation schreiben
        - Für Benutzung von vanilla-extract in Sols und inSuite
    - Webpack/ESBuild müssen plugins installiert werden.

## Technical requirements styleguide documentation tool

### requisites for the documentation framework

- Support dynamic components (React, Vue, Svelte, etc.)
